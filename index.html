Inferencia de animalitos por medio de una IA 🦋🦊🦓

¡Bienvenido!

Este es una inteligencia articial capaz de identificar, utilizando una imagen como referencia, varias especies de animales.

Para comenzar, carga una imagen utilizando el botón Subir Archivo, y oprime el botón Iniciar Predicción para empezar con el reconocimiento.

No se ha seleccionado ninguna imagen.

Animales que puede detectar la IA: Ajolote - Águila Calva - Águila Real - Luciernága - Mariposa Monarca - Oso Polar - Zorro - León Marino - Vaquita Marina - Zebra
